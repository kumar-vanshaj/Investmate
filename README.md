# 📱 InvestX – FinTech Mobile App

InvestX is a cross-platform FinTech mobile application built using Flutter that enables users to track stocks, manage portfolios, and visualize market trends.

## 🚀 Features (Planned)

* User Authentication (JWT/Firebase)
* Real-time Stock Data Integration
* Portfolio Tracking
* Watchlist Management
* Interactive Charts (fl_chart)

## 🧱 Tech Stack

* Flutter (UI)
* Provider (State Management)
* REST APIs (Stock Data)
* Firebase / Node.js (Backend - planned)
